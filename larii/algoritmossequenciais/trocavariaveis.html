Troca de Variáveis

Troca de valores entre duas variáveis.

Exemplo de Código JavaScript:

let a = 5;
let b = 10;
console.log("Antes da troca:");
console.log("a =", a, "b =", b);

// Troca dos valores
let temp = a;
a = b;
b = temp;

console.log("Após a troca:");
console.log("a =", a, "b =", b);
