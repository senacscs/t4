Problema de Pesquisa

Como promover padrões de consumo e produção sustentáveis para reduzir o desperdício, minimizar o impacto ambiental e garantir o uso eficiente dos recursos naturais?

Soluções existentes

Já existem várias plataformas como OLX, Mercado Livre e eBay que permitem a venda de produtos usados

Link para o site do Mercado Livre: mercadolivre.com.br

Link para o site da OLX: olx.com.br

Link para o site do eBay: br.ebay.com

Referências

Criar um site de vendas online para a pessoa vender o seu produto que não está mais utilizando mais pode ser útil para outra pessoa.

Objetivo do Porjeto

A criação de um site de vendas online para produtos usados (brecho digital) aumentará a reutilização de itens, reduzirá o desperdício e proporcionará benefícios econômicos para vendedores e compradores.
