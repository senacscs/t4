Pseudo-elementos CSS - Exemplos

1. ::first-line

Esta linha ficará com destaque. O restante do parágrafo mantém o estilo padrão. Você pode ver a diferença ao redimensionar a janela.

2. ::first-letter

A primeira letra deste parágrafo será maior e colorida, simulando o estilo de livros antigos ou revistas.

3. ::before

Este parágrafo possui um símbolo adicionado antes do seu conteúdo com ::before.

4. ::after

Este parágrafo adiciona um ícone no final usando ::after.

5. ::selection

Selecione este texto com o mouse para ver o efeito do pseudo-elemento ::selection.
