⭐Tutorial textual⭐

HTML básico:

  • HTML é o código que você usa para estruturar uma página web e seu conteúdo.
  • HTML consiste de uma série de elementos, que você usa para delimitar ou agrupar diferentes partes do conteúdo para que ele apareça ou atue de determinada maneira.

TUTORIAL

principais partes de um elemento:

  • TAG DE ABERTURA: Consiste no nome do elemento envolvido em parênteses angulares de abertura e fechamento. Demonstra onde o elemento começa.
  • TAG DE FECHAMENTO: Consiste na mesma coisa que a tag de abertura, exceto que inclui uma barra antes do nome do elemento. Demonstra onde o elemento acaba.
  • CONTEÚDO: Esse é o conteúdo do elemento que, nesse caso, é apenas um texto.
  • ELEMENTO: A tag de abertura, a tag de fechamento e o conteúdo formam o elemento.
