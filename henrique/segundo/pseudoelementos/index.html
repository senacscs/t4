Pseudo-elementos CSS - Exemplos

1. p::first-line

A primeira linha deste texto será estilizada com uma cor diferente para destacar seu conteúdo inicial.

2. p::first-letter

Veja como a primeira letra deste parágrafo é destacada, simulando o estilo de início de capítulos de livros.

3. p::before

Este texto recebe um símbolo antes de ser exibido, apenas com CSS, sem alterar o HTML diretamente.

4. p::after

Um ícone aparece no final desta frase como se fosse parte do conteúdo, mas é inserido pelo CSS.

5. p::selection

Ao selecionar este parágrafo com o mouse, o fundo mudará de cor, graças ao uso do ::selection.
