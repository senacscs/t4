O que é o CSS?

  • CSS é o código que você usa para dar estilo à sua página Web.

TUTORIAL

principais partes de um elemento:

  • SELECTOR: Seleciona os elementos a serem estilizados, nesse caso, elementos "p"
  • DECLARATION: Especifica quais das propriedades do elemento você quer estilizar, nesse caso, "color: red;".
  • PROPERTY: Forma pela qual você estiliza um elemento HTML, nesse caso, "color" é uma propriedade dos elementos "p".
  • PROPERTY VALUE: Escolhe uma dentre muitas aparências possíveis para uma determinada propriedade, há muitos valores "color" além do "red".
