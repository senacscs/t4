Jogo de Enigmas - Funções Matemáticas

Enigma 1

Resolva o enigma para avançar:

Qual é a raiz da equação x² - 5x + 6 = 0?

Enigma 2

Resolva o enigma para avançar:

Qual é a soma das raízes da equação 2x² - 7x + 3 = 0?

Enigma 3

Resolva o enigma para avançar:

Qual é o vértice da parábola descrita pela função f(x) = x² + 4x - 5?

Enigma 4

Resolva o enigma para avançar:

Uma função quadrática f é dada por f(x) = x2 + bx + c, com b e c reais. Se f(1) = –1 e f(2) – f(3) = 1, o menor valor que f(x) pode assumir, quando x varia no conjunto dos números reais, é igual a 3x - 4 = 0?

Enigma 5

Resolva o enigma para avançar:

Qual é o valor de f(3) para a função f(x) = 2x - 1?

Enigma 6

Resolva o enigma para avançar:

Qual é o valor de x para o qual a função g(x) = -2x² + 3x + 1 atinge seu valor máximo?

Enigma 7

Resolva o enigma para avançar:

Para que valor de x a função h(x) = 4x - x² é crescente?

Enigma 8

Resolva o enigma para finalizar o jogo:

Qual é a solução da equação x² - 6x + 8 = 0?
