1. O que é HTML?

HTML (Hyper Text Markup Language) é a linguagem usada para estrutar páginas web

2. Estrutura básica do HTML

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML</title>
</head>
<body>
     <h1> Olá, Mundo!</h1>
</body>
</html>
