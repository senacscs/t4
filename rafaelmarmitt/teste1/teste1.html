Revisão: HTML e CSS

1. Estrutura HTML

O HTML (Hyper Text Markup Language) é a linguagem utilizada para estruturar páginas web.

  • As páginas são compostas por elementos como <head> e <body>.
  • Os elementos podem conter textos, imagens, tabelas e links.

2. Imagens

O HTML permite a incerção de imagens usando a tag <img>
