Aprenda toda base do CSS de forma online

CSS (Cascading Style Sheets) é uma linguagem que descreve como os elementos HTML devem ser apresentados na tela, permitindo que você controle o layout, as cores, as fontes e outros aspectos visuais de uma página web.




A primeira indicação é o site FreeCodeCamp

O FreeCodeCamp é uma plataforma de aprendizado online que oferece cursos gratuitos de programação e desenvolvimento web para iniciantes e programadores experientes. Com mais de 40.000 alunos, o FreeCodeCamp é uma das maiores comunidades de aprendizado de programação do mundo.
