Demonstração de Pseudo-elementos CSS

Esta página demonstra o uso dos pseudo-elementos CSS abordados em aula.

::first-line

Este pseudo-elemento aplica estilos à primeira linha de um elemento de bloco. Observe como apenas a primeira linha deste parágrafo tem aparência diferente das demais. O estilo é aplicado dinamicamente conforme o tamanho da janela é alterado. Se você redimensionar o navegador, verá que a formatação da primeira linha se ajusta automaticamente.

::first-letter

Este pseudo-elemento aplica estilos à primeira letra de um elemento de bloco. É comumente usado para criar efeitos de letra capitular, como em livros e revistas. A primeira letra deste parágrafo foi ampliada e estilizada para se destacar visualmente, criando um ponto focal interessante para o texto.

::before e ::after

Estes pseudo-elementos permitem inserir conteúdo antes ou depois do conteúdo de um elemento. Eles são frequentemente usados para adicionar ícones, elementos decorativos ou texto adicional sem modificar o HTML. Neste exemplo, adicionamos aspas decorativas no início e no final deste bloco de texto.

::selection

Este pseudo-elemento aplica estilos ao texto selecionado pelo usuário. Selecione parte deste texto para ver o efeito. Em vez do padrão (geralmente azul com texto branco), configuramos uma cor roxa com texto branco para a seleção. Isso pode ser usado para melhorar a experiência do usuário e manter a consistência com a identidade visual do site.

Pseudo-elementos são poderosas ferramentas CSS que permitem estilizar partes específicas dos elementos sem a necessidade de adicionar markup extra ao HTML.
