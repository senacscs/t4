Exemplos de Tipos de Display CSS

display: block

Elemento Block 1
Elemento Block 2

Elementos block ocupam toda a largura disponível e iniciam em uma nova linha. Eles também respeitam propriedades de largura e altura.

display: inline

Texto antes do Elemento Inline 1 e depois Elemento Inline 2.

Elementos inline ocupam apenas o espaço necessário para seu conteúdo e não iniciam em uma nova linha. Largura e altura não têm efeito.

display: inline-block

Inline-Block 1
Inline-Block 2

Elementos inline-block se comportam como inline no fluxo, mas permitem definir largura e altura, além de respeitar margens e paddings verticais.

display: none

Este texto contém um elemento ESCONDIDO.

Elementos com display: none não são exibidos e não ocupam espaço no layout. É diferente de `visibility: hidden`, que ocupa o espaço mas não é visível.
