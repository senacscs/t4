O que é o HTML

O código conhecido como HTML (Linguagem de Marcação de Hipertexto) é usado para organizar uma página web e seu conteúdo. Embora seja uma linguagem de marcação, não é uma linguagem de programação. Vamos examinar os fundamentos do HTML.

Definição e Estrutura:

  • A estrutura do conteúdo de uma página pode ser definida usando o HTML.
  • Você pode organizar o conteúdo em tabelas, parágrafos, listas com marcadores e imagens.
  • Ele é composto por uma variedade de componentes que separam ou organizam várias partes do conteúdo para que ele apareça ou funcione de forma específica.

Anatomia de um Elemento HTML:

  • Tag de Abertura: Inclui o nome do elemento (por exemplo, "p" para um parágrafo) e indica onde começa o elemento.
  • Tag de Fechamento: Tem uma barra antes do nome do elemento, mas é semelhante à tag de abertura. Indica a conclusão do elemento.
  • Conteúdo: É o texto ou outros elementos dentro do elemento.
  • Atributos: Informações extras sobre o elemento (como o "class" para aplicar estilos).
