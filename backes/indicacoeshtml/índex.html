O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criação de páginas web. Com HTML, podemos estruturar o conteúdo de uma página, como textos, imagens, vídeos, links, etc.

Ele é usado para definir a estrutura da web, e é a base para qualquer página, junto com CSS (Cascading Style Sheets) e JavaScript.
