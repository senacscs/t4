Indicações de HTML

HTML é uma linguagem de marcação usada para estruturar uma página web. Composto por elementos, é possível anexar, envolver ou marcar partes do conteúdo.

As tags permitem transformar esse conteúdo em um link ou colocar palavras em itálico, entre outras ações. Exemplo:

"A Terra é redonda"

Para a linha aparcer precisamos especificar que ela é um parágrafo, utilizando "< p >". Exemplo:

< p > A Terra é redonda < / p >"

Principais partes do elemento:

Tag de abertura (p),

tag de fechamento (/p)

conteúdo (texto).

A tag de abertura indica onde o elemento começa, a tag de fechamento indica onde termina e o conteúdo é o que está dentro do elemento.
