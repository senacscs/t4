CSS (Cascading Style Sheets) é uma linguagem de folhas de estilo usada para descrever a aparência e formatação de um documento escrito em HTML. CSS é uma das principais tecnologias do núcleo da Web e é usado para estilizar páginas web e interfaces de usuário escritas em HTML e XHTML. Pode também ser aplicado a qualquer tipo de documento XML, incluindo SVG e XUL.

O CSS é uma ferramenta poderosa que permite aos desenvolvedores controlar precisamente como os elementos HTML são exibidos na tela, melhorando a experiência do usuário e a acessibilidade do site. Ele é uma parte essencial do desenvolvimento web front-end, juntamente com HTML e JavaScript.
