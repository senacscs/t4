Conectando a um servidor

O status de conexão do Relatório da web para um servidor da Internet aparece no canto superior direito do Relatório da web.
Uma conexão com a Internet não é obrigatória para abrir um Relatório da web. Com uma cópia salva do Relatório da web, é possível visualizar todos os gráficos e tabelas e ter alguma capacidade de edição limitada quando não estiver conectado à Internet.
Conectando um Relatório da web para um servidor da Internet ativará muito mais recursos de edição para tabelas e para gráficos.
  • Se o autor tiver especificado um servidor da Internet ao criar o Relatório da web, o Relatório da web tentará se conectar ao servidor automaticamente quando for aberto.
  • Se o Relatório da web não se conectar a um servidor, clique na Mensagem de status do servidor para abrir ferramentas para tentar novamente a conexão, tentar um servidor diferente ou inserir um novo endereço do servidor.
  • Para obter informações sobre como incluir os controles aprimorados em seu servidor da Internet, acesse https://developer.ibm.com/predictiveanalytics.
  • Se especificar uma nova conexão do servidor, o formato preferencial será http://xxx.xxx.xxx.xxx:xxxx.
