Você chegou ao universo de The Society

Um grupo de adolescentes do ensino médio fica preso na cidade de New Ham, onde todas as saídas estão bloqueadas por árvores. Ao investigar, descobrem que cada árvore tem uma equação a ser resolvida. Cada adolescente escolhe uma árvore e, à medida que as equações são solucionadas, as passagens para fora da cidade começam a se desbloquear. Resolva esta equação para liberar mais uma passagem ( x + 3 = 7).

Qual das opções é a resposta correta para essa conta?
x = 4
x = 9
x = 2
x = 5

Ajude-os a desbloquearem as passagens!
