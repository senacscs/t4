Você passou para a Cabana do Mistério!

Dipper e Mabel estão explorando uma floresta misteriosa quando encontram um diário escondido. Para abrir o diario, eles precisam resolver um enigma que diz: x² + 2x - 3 = 0.

Qual das opções é a resposta correta para essa conta?
x = 2 ou x = -3
x = 4 ou x = -5
x = 1 ou x = -3
x = 5 ou x = -2

Ajude Dipper e Mabel a resolverem o enigma e abrirem o portal para um mundo cheio de mistérios e aventuras!
