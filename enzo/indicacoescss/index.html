O que é CSS?

CSS, ou Cascading Style Sheets, é uma linguagem de estilo usada para descrever a apresentação de documentos escritos em HTML ou XML. CSS é uma das principais tecnologias da World Wide Web, ao lado do HTML e do JavaScript.
Alguns pontos-chave sobre o CSS:
Separação de Conteúdo e Estilo:
CSS permite separar o conteúdo de um documento (HTML) de sua apresentação (como o documento é estilizado e apresentado visualmente).
Seletores:
CSS usa seletores para associar estilos a elementos HTML. Seletores podem ser simples, como selecionar todos os elementos de um tipo específico, ou complexos, como selecionar elementos baseados em suas classes, IDs, atributos, e relações com outros elementos.
Propriedades e Valores:
CSS aplica estilos aos elementos usando propriedades e valores. Por exemplo, color: red; aplica a cor vermelha ao texto, e margin: 10px; define uma margem de 10 pixels ao redor de um elemento.
Cascata e Especificidade:
O termo "cascading" se refere a como as regras de estilo são aplicadas. Quando múltiplas regras se aplicam a um elemento, a "cascata" e a "especificidade" determinam quais estilos prevalecerão. Regras mais específicas e aquelas definidas mais tarde geralmente têm precedência.
Layouts Responsivos:
CSS permite criar layouts responsivos que se adaptam a diferentes tamanhos de tela e dispositivos. Ferramentas como media queries são usadas para aplicar estilos diferentes com base nas características do dispositivo de visualização, como largura da tela.
Animações e Transições:
CSS suporta animações e transições que permitem criar efeitos visuais dinâmicos, como mudar suavemente a cor de um botão ao passar o cursor sobre ele.
