A página "CSS Básico" do MDN Web Docs é um guia introdutório que explica os fundamentos do CSS, uma linguagem usada para estilizar documentos HTML. Ela cobre os principais conceitos, como seletores, propriedades e valores, e como aplicar estilos a elementos HTML. A seção também inclui exemplos práticos para ajudar os iniciantes a aprender a usar CSS para criar páginas web visualmente atraentes. Acesse o mozilla developer aqui!
