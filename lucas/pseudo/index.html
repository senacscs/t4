Exemplos Combinadores CSS

Combinador de Filho Direto (>)

Seleciona elementos que são filhos diretos de outro elemento.

Filho direto (selecionado)

Parágrafo dentro do span (não selecionado)

Combinador de Descendente (espaço)

Seleciona todos os elementos descendentes, independentemente do nível.

Descendente (selecionado)

Outro descendente (selecionado)

Combinador de Irmão Adjacente (+)

Seleciona o próximo irmão imediato de um elemento.

Elemento Base

Irmão direto (selecionado)

Outro irmão (não selecionado)

Combinador de Irmãos Seguintes (~)

Seleciona todos os irmãos que vêm depois do elemento base.

Elemento base

Primeiro irmão (selecionado)

Segundo irmão (selecionado)
